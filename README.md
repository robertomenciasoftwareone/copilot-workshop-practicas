# Copilot Workshop - Prácticas

Este repositorio contiene ejercicios prácticos para aprender a usar GitHub Copilot.

## Contenido

- **ejercicios/**: Carpeta con ejercicios prácticos
- **soluciones/**: Carpeta con soluciones de ejemplo

## Ejercicios

1. **Básicos**: Funciones simples y algoritmos básicos
2. **Intermedio**: Estructuras de datos y manipulación de archivos
3. **Avanzado**: APIs, testing y optimización

## Cómo usar este repositorio

1. Clona el repositorio
2. Abre cada ejercicio en VS Code con GitHub Copilot activado
3. Intenta completar los ejercicios usando Copilot como asistente
4. Compara tu solución con las propuestas en la carpeta de soluciones

## Requisitos

- Visual Studio Code
- GitHub Copilot extension
- Node.js (para ejercicios de JavaScript/TypeScript)
- Python 3.x (para ejercicios de Python)
